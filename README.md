# Cuba Tattoo Studio

Official website for Cuba Tattoo Studio - Art, Passion & Tradition.

## 🎨 Description

Static website for Cuba Tattoo Studio, a professional tattoo studio that combines traditional Cuban art with modern techniques.

## 🚀 Technologies

- **HTML5** - Semantic structure
- **CSS3/SCSS** - Responsive styling
- **JavaScript ES6+** - Interactivity
- **GSAP 3.x** - Professional animations
- **Vite** - Development tooling

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/terrerovgh/cubatattoostudio.git

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 🌐 Deployment

The site is automatically deployed to GitHub Pages.
