# 🇨🇺 Cuba Tattoo Studio

**Arte Auténtico Cubano en Albuquerque, Nuevo México**

## 🎨 Sobre Nosotros

Cuba Tattoo Studio es más que un estudio de tatuajes - somos guardianes de la rica tradición artística cubana. Desde 2020, nuestros artistas han combinado técnicas modernas con la herencia cultural cubana para crear obras únicas que honran tu historia y expresan tu individualidad.

## ✨ Características del Sitio Web

- **Diseño Cinematográfico**: Animaciones fluidas con GSAP y ScrollTrigger
- **Temática Cubana**: Paleta de colores inspirada en la bandera cubana (rojo, azul, dorado)
- **Responsive Design**: Optimizado para todos los dispositivos
- **Efectos Visuales**: Transiciones elegantes y partículas flotantes
- **Performance**: Carga rápida y experiencia suave

## 🚀 Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Animaciones**: GSAP 3.12.2 con ScrollTrigger
- **Build Tool**: Vite 6.3.5
- **Hosting**: GitHub Pages (próximamente)

## 🛠️ Desarrollo

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/terrerovgh/cubatattoostudio.git

# Navegar al directorio
cd cubatattoostudio

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

### Scripts Disponibles

- `npm run dev` - Servidor de desarrollo
- `npm run build` - Build para producción
- `npm run preview` - Preview del build
