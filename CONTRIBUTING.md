# Contributing to Cuba Tattoo Studio

We welcome contributions to improve the project!

## Workflow

1.  **Fork the repository**.
2.  **Create a branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
3.  **Make your changes**.
4.  **Commit your changes** with descriptive messages.
5.  **Push to your branch**:
    ```bash
    git push origin feature/amazing-feature
    ```
6.  **Open a Pull Request**.

## Guidelines

- **Code Style**: Follow the existing code style. Run `npm run format` before committing.
- **Testing**: Add tests for new features if possible. Ensure existing tests pass.
- **Documentation**: Update documentation if you change functionality or add new features.

## Reporting Issues

If you find a bug or have a suggestion, please open an issue in the repository with details about the problem or idea.
