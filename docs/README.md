## Convenciones de Nomenclatura

Para mantener esta carpeta organizada, por favor sigue estas convenciones al añadir documentos:

1. Usa nombres descriptivos para los archivos
2. Utiliza guiones para separar palabras en los nombres de archivo (ej. `guia-estilos.md`)
3. Incluye la fecha en formato YYYY-MM-DD para documentos que evolucionan con el tiempo
4. Utiliza Markdown (.md) para documentos de texto cuando sea posible
5. Organiza los archivos en subcarpetas según las categorías mencionadas arriba
