# Cuba Tattoo Studio - Project Documentation

Welcome to the documentation for the Cuba Tattoo Studio web application. This project is a modern, high-performance website built with Astro and React, featuring an AI-powered assistant capable of running entirely in the browser.

## Documentation Sections

*   **[Architecture](./ARCHITECTURE.md)**: Overview of the technology stack, project structure, and key architectural decisions.
*   **[Data Models (API)](./API.md)**: Details on the Content Collections schema and data organization.
*   **[AI System](./AI.md)**: In-depth explanation of the client-side AI assistant, including model details, Web Worker implementation, and the booking flow.
*   **[Deployment](./DEPLOYMENT.md)**: Instructions for building, configuring, and deploying the application.
*   **[Future & Recommendations](./FUTURE.md)**: Roadmap for future features, optimizations, and considerations.

## Quick Start

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Download AI Model** (Required for the AI features to work locally):
    ```bash
    npm run download-model
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```
