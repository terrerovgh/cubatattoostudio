# Documentación Suplementaria

Esta carpeta contiene guías suplementarias para el proyecto Cuba Tattoo Studio.

## 📁 Contenido

### [content-management.md](./content-management.md)
Guía para gestión de contenido sin conocimientos técnicos profundos:
- Actualizar imágenes (artistas, galería)
- Modificar texto en componentes
- Añadir/eliminar artistas
- Actualizar información de contacto
- Cambiar logo y favicon

### [animation-guide.md](./animation-guide.md)
Documentación completa del sistema de animaciones:
- Reveal animations con Intersection Observer
- Efecto parallax del hero
- Hover effects coordinados
- Personalización de animaciones
- Performance y optimización

### [styling-guide.md](./styling-guide.md)
Guía exhaustiva del sistema de estilos con TailwindCSS 4:
- Configuración de Tailwind v4
- Paleta de colores y tipografía
- Spacing y layout patterns
- Responsive design
- Custom CSS y mejores prácticas

## 🎯 Para Quién Es Cada Guía

| Guía | Audiencia | Nivel Técnico |
| :--- | :--- | :--- |
