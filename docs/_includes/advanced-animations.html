Magnetic Hover

Interactive magnetic effect that follows mouse movement with elastic return.

HOVER
üìã View Code ```javascript // Magnetic Hover Effect function createMagneticEffect(element) { element.addEventListener('mousemove', (e) => { const rect = element.getBoundingClientRect(); const x = e.clientX - rect.left - rect.width / 2; const y = e.clientY - rect.top - rect.height / 2; gsap.to(element, { duration: 0.3, x: x * 0.3, y: y * 0.3, ease: 'power2.out' }); }); element.addEventListener('mouseleave', () => { gsap.to(element, { duration: 0.5, x: 0, y: 0, ease: 'elastic.out(1, 0.3)' }); }); } // Apply to elements document.querySelectorAll('.magnetic').forEach(createMagneticEffect); ``` ```css .magnetic { cursor: pointer; transition: transform 0.1s ease; } .magnetic:hover { transform: scale(1.05); } ```
