# Blueprint & Protocolo de Desarrollo

## Visión y Objetivos
Crear el centro digital de Cubatattoo Studio, combinando estética artística y funcionalidad para atraer y convertir clientes exigentes.

## Público Objetivo
"El Coleccionista Exigente": Adultos de 25-45 años, conocedores del arte, buscan profesionalismo y experiencia impecable.

## Estética de Marca
- Modern Grit: combinación de cultura clásica del tatuaje y diseño digital moderno.
- Uso de fotografía dramática, espacio en blanco, colores audaces y tipografía fuerte.

## Arquitectura Técnica
- Framework: Next.js (SSR/SSG)
- Lenguaje: TypeScript
- Estilos: Tailwind CSS
- Componentes visuales: reactbits.dev

## Fases de Implementación
1. Crear proyecto Next.js con TypeScript y Tailwind.
2. Instalar dependencias principales (`framer-motion`, `gsap`).
3. Integrar componentes reactbits.dev siguiendo el protocolo AIOp.
4. Implementar páginas y funcionalidades por fases.

## Protocolo de Componentes (AIOp)
- Navegar a reactbits.dev, copiar código y dependencias.
- Crear archivo en `/src/components/react-bits/`.
- Importar y usar en la página correspondiente.
