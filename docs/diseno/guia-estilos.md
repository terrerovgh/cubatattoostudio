# Guía de Estilos - Cuba Tattoo Studio

Esta guía define los estándares visuales y de diseño para el sitio web de Cuba Tattoo Studio, asegurando consistencia en toda la plataforma.

## Paleta de Colores

### Colores Primarios

- **Rojo Cubano:** `#D62828` - Color principal, representa la pasión y la energía del estudio
- **Azul Caribe:** `#003049` - Color secundario, evoca el mar caribeño y aporta profesionalismo
- **Crema Habana:** `#FCBF49` - Acento cálido, reminiscente de la arquitectura colonial cubana

### Colores Secundarios

- **Negro Tinta:** `#0A0908` - Para textos principales y elementos de contraste
- **Blanco Lienzo:** `#F8F9FA` - Fondo principal y espacios negativos
- **Gris Neutro:** `#6C757D` - Textos secundarios y elementos de interfaz

### Colores de Acento

- **Verde Palma:** `#2A9D8F` - Acento para elementos de éxito o confirmación
- **Naranja Tropical:** `#F77F00` - Llamadas a la acción y elementos destacados

## Tipografía

### Títulos

- **Familia:** "Playfair Display" (serif)
- **Pesos:** Regular (400), Bold (700)
- **Usos:** Títulos principales, encabezados de sección, nombres de artistas

### Cuerpo de Texto

- **Familia:** "Montserrat" (sans-serif)
- **Pesos:** Light (300), Regular (400), Medium (500), Semi-Bold (600)
- **Usos:** Párrafos, descripciones, información de contacto, navegación

### Acentos Tipográficos

- **Familia:** "Permanent Marker" (display)
- **Usos:** Elementos decorativos, frases destacadas, efectos especiales

## Iconografía

- Estilo de línea consistente con grosor medio
- Esquinas ligeramente redondeadas
- Paleta monocromática adaptada a la sección
- Set recomendado: Font Awesome o similar

## Elementos de Diseño

### Botones

- **Primario:** Fondo `#D62828`, texto blanco, bordes redondeados (8px)
- **Secundario:** Borde `#003049`, texto `#003049`, fondo transparente
- **Terciario:** Texto `#D62828`, sin borde, subrayado en hover

### Tarjetas y Contenedores

- Sombras suaves (4px blur, 30% opacidad)
- Bordes redondeados (12px)
- Fondo blanco o muy claro
- Acentos de color en bordes o elementos internos

### Imágenes y Fotografías

- Tratamiento consistente con filtro cálido sutil
- Contraste aumentado ligeramente
- Bordes redondeados o máscaras artísticas para thumbnails
- Relación de aspecto consistente por categoría

## Animaciones (anime.js)

### Principios Generales

- Animaciones suaves y naturales
- Duración: 300-500ms para transiciones estándar
- Curvas de aceleración: easeOutExpo para la mayoría de las animaciones
- No interrumpir la experiencia del usuario

### Animaciones Específicas

- **Scroll Reveal:** Elementos que aparecen sutilmente al hacer scroll
- **Hover Effects:** Cambios suaves en escala (1.05x) y sombra
- **Página de Inicio:** Animación destacada de máquina de tatuar
- **Transiciones de Página:** Desvanecimiento suave entre secciones

## Estilo Fotográfico

- **Estilo:** Alto contraste, tonos cálidos, iluminación dramática
- **Contenido:** Trabajo de tatuajes, artistas en acción, detalles del estudio
- **Tratamiento:** Consistente en toda la web, posible viñeteado sutil
- **Fondos:** Neutros o desenfocados para destacar el sujeto principal

## Elementos de Identidad Cubana

- Patrones geométricos inspirados en azulejos cubanos
- Motivos de palmas y elementos tropicales como separadores
- Referencias sutiles a la bandera cubana en elementos decorativos
- Texturas que evoquen la arquitectura de La Habana

---

## Aplicación Responsive

### Breakpoints

- **Móvil:** < 576px
- **Tablet:** 576px - 992px
- **Desktop:** > 992px

### Consideraciones Móviles

- Menú hamburguesa con animación suave
- Tipografía aumentada para mejor legibilidad
- Elementos touch-friendly (min. 44px)
- Simplificación de animaciones complejas

---

Esta guía está sujeta a evolución a medida que el proyecto avance. Cualquier modificación debe ser documentada y comunicada al equipo.
